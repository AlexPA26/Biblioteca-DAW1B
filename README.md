# BIBLIOTECA JAVA PROYECTO MUY ÉPICO Y FUNCIONAL

## INTRODUCCIÓN

Este proyecto java trata de crear una biblioteca agregando libros y autores para que aparezcan en la terminal de manera sencilla y organizada.

**El programa contiene lo siguiente:**
+ Puedes agregar el título y autor
+ Utilizada arrays para contar los libros del inventario
+ Organización de clases profesional y organizada
